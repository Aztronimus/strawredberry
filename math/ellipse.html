Ellipse

\begin{align} a + b = d - c \tag{1} \end{align}

Such that \(d > 2c\).

\begin{align} a^2 = \left( \frac{c}{2} + x \right)^2 + y^2 \tag{2} \end{align} \begin{align} b^2 = \left( \frac{c}{2} - x \right)^2 + y^2 \tag{3} \end{align}

From \((1)\).

\begin{align} (a + b)^2 = (d - c)^2\\ \end{align} \begin{align} a^2 + b^2 + 2ab = (d - c)^2 \tag{4} \end{align}

From the addition and substraction of \((2)\) and \((3)\).

\begin{align} a^2 - b^2 = (a + b)(a - b) = 2cx \tag{5} \end{align} \begin{align} a^2 + b^2 = \frac{c^2}{2} + 2x^2 + 2y^2 \tag{6} \end{align}

Substituting \((1)\) into \((5)\) and re-arranging.

\begin{align} (a + b)(a - b) = (d - c)(a - b) = 2cx \end{align} \begin{align} a - b = \left( \frac{2c}{d - c} \right) x \end{align} \begin{align} (a - b)^2 = a^2 + b^2 - 2ab = \left(\frac{2c}{d - c}\right)^2 x^2 \tag{7} \end{align}

From the addition of \((4)\) and \((7)\).

\begin{align} 2 \left(a^2 + b^2\right) = \left(\frac{2c}{d - c}\right)^2 x^2 + (d - c)^2 \tag{8} \end{align}

Substituting \((6)\) into \((8)\) and re-arranging.

\begin{align} c^2 + 4x^2 + 4y^2 = \left(\frac{2c}{d - c}\right)^2 x^2 + (d - c)^2 \end{align} \begin{align} \left(4 - \left(\frac{2c}{d - c}\right)^2 \right)x^2 + 4y^2 &= (d - c)^2 - c^2 \end{align} \begin{align} \left( \frac{4d^2 + 4c^2 - 8dc - 4c^2}{(d - c)^2} \right)x^2 + 4y^2 &= d^2 - 2dc \end{align} \begin{align} \left( \frac{4(d^2 - 2dc)}{d^2 - 2dc + c^2} \right)x^2 + 4y^2 &= d^2 - 2dc \end{align} \begin{align} \left( \frac{4}{d^2 - 2dc + c^2} \right) x^2 + \left( \frac{4}{d^2 - 2dc}\right )y^2 &= 1 \tag{9} \end{align}

Which is of the form:

\begin{align} \frac{x^2}{c_x^2} + \frac{y^2}{c_y^2} &= 1 \end{align}

With:

\begin{align} c_x^2 &= \frac{d^2 - 2dc + c^2}{4} & c_y^2 &= \frac{d^2 - 2dc}{4} \end{align}

Notice that the focus of the ellipse can be obtained using:

\begin{align} \frac{c}{2} = \sqrt{c_x^2 - c_y^2} \end{align}

Perimeter

Arc length

\begin{align} s = \int_{x_0}^{x_1}{ \sqrt{ 1 + \left( \frac{dy}{dx} \right)^2} }dx \end{align} \begin{align} y &= c_y \sqrt{1 - \frac{x^2}{c_x^2}} = \frac{c_y}{c_x} \sqrt{c_x^2 - x^2}\\ \frac{dy}{dx} &= - \frac{c_y}{c_x} \cdot \frac{x}{\sqrt{c_x^2 - x^2}}\\ \left( \frac{dy}{dx} \right)^2 &= \frac{c_y^2}{c_x^2} \cdot \frac{x^2}{c_x^2 - x^2} = \frac{k^2 x^2}{c_x^2 - x^2} \end{align} \begin{align} k^2 = \frac{c_y^2}{c_x^2} \end{align} \begin{align} p &= 4\int_{0}^{c_x}{ \sqrt{ 1 + \frac{k^2 x^2}{c_x^2 - x^2} } } \cdot dx = 4\int_{0}^{c_x}{ \sqrt{ \frac{c_x^2 - (1 - k^2) x^2}{c_x^2 - x^2} } } \cdot dx\\ \end{align}

Taylor series

\begin{align} q(x) &= \sqrt{ 1 + \left( \frac{dy}{dx} \right)^2}\\ &= \sum_{n=0}^\infty { \frac{(x - x_0)^n}{n!} \cdot \left. \frac{d^nq}{dx^n} \right|_{x_0} } \end{align} \begin{align} p = 4 \int_{0}^{c_x} q(x) dx = \sum_{n=0}^\infty { \frac{x^{n + 1}}{(n + 1)!} \cdot \left. \frac{d^nq}{dx^n} \right|_{x_0} } \end{align}

To get the \(n\) derivative of \(q\):

\begin{align} h(x) &= \left( \frac{dy}{dx} \right)^2 \end{align} \begin{align} \frac{d^n q}{dx^n} &= \frac{d^{n - 1}}{dx^{n - 1}} \left( \frac{dq}{dh} \cdot \frac{dh}{dx} \right) \end{align} \begin{align} \frac{d^m}{dh^m} \left( \frac{1}{h^n} \right) &= \frac{d^{m - 1}}{dh^{m - 1}} \left( \frac{d}{dh} \left( \frac{1}{h^n} \right) \right) = \frac{d^{m - 1}}{dh^{m - 1}} \left( - \frac{n}{h^{n + 1}} \right)\\ &= - n \cdot \frac{d^{m - 1}}{dh^{m - 1}} \left( \frac{1}{h^{n + 1}} \right) \end{align}

Now the induction hypothesis (for \(m\)) is:

\begin{align} \frac{d^m}{dh^m} \left( \frac{1}{h^n} \right) &= \left( \prod_{j = 0}^{m - 1} - n - j \right) \frac{1}{h^{n + m}} \end{align}

By induction hypothesis (for \(m + 1\)) is:

\begin{align} \frac{d^{m + 1}}{dh^{m + 1}} \left( \frac{1}{h^n} \right) &= - n \cdot \frac{d^m}{dh^m} \left( \frac{1}{h^{n + 1}} \right)\\ &= -(n + 1) \cdot \left( \prod_{j = 0}^{m - 1} - n - 1 - j \right) \frac{1}{h^{n + m + 1}} \end{align}

Coming back to the problem:

\begin{align} \frac{d^n q}{dh^n} &= \frac{d^{n-1}}{dh^{n-1}} \left( \frac{dq}{dh} \right) = \frac{d^{n-1}}{dh^{n-1}} \left( \frac{1}{2h} \right)\\ &= \frac{1}{2} \cdot \frac{d^{n-1}}{dh^{n-1}} \left( \frac{1}{h} \right) \end{align}

To get the \(n\) derivative of \(y\):

\begin{align} y &= \frac{c_y}{c_x} \cdot f(x) & f(x) &= \sqrt{c_x^2 - x^2} \end{align} \begin{align} \frac{d^ny}{dx^n} &= \frac{c_y}{c_x} \cdot \frac{d^nf}{dx^n} \end{align} \begin{align} \frac{df}{dx} &= - \frac{x}{\sqrt{c_x^2 - x^2}} = - \frac{x}{f}\\ \frac{d^2f}{dx^2} &= \frac{x}{f^2} - \frac{1}{f}\\ \end{align}

We can get a pattern from this point.

\begin{align} \frac{d}{dx}\left( \frac{1}{f^n} \right) &= - \frac{n}{f^{n + 1}} & \frac{d}{dx}\left( \frac{x}{f^n} \right) &= \frac{1}{f^n} - \frac{nx}{f^{n + 1}} \end{align} \begin{align} \frac{d}{dx}\left( \frac{x}{f^n} - \frac{1}{f^{n - 1}} \right) &= \frac{d}{dx}\left( \frac{x}{f^n} \right) - \frac{d}{dx}\left( \frac{1}{f^{n - 1}} \right)\\ &= \frac{1}{f^n} - \frac{nx}{f^{n + 1}} + \frac{n - 1}{f^n}\\ &= n \left( \frac{1}{f^n} - \frac{x}{f^{n + 1}} \right) \end{align}

\(m\) derivative formula for \(g_n\).

\begin{align} g_n(x) &= \frac{x}{f^n} - \frac{1}{f^{n - 1}} & \frac{d^m g_n}{dx^m} &= - n \cdot \frac{d^{m - 1} g_{n + 1}}{dx^{m - 1}} \end{align}

Which are only valid for \(n \ge 2\).

\begin{align} \prod_{j=m}^n a_j &= a_m \cdot \left( \prod_{j = m + 1}^n a_j \right) & \prod_{j=m}^m a_j &= a_m & \prod_{j=m}^n a_j &= \left( \prod_{j=m}^{n - 1} a_j \right) \cdot a_n \end{align}

Induction hypothesis (for \(k\)).

\begin{align} \frac{d^m g_n}{dx^m} &= \left( \prod_{j = 0}^{k - 1} - n - j \right) \frac{d^{m - k} g_{n + k}}{dx^{m - k}} \end{align}

By induction hypothesis (for \(k + 1\)).

\begin{align} \frac{d^m g_n}{dx^m} &= \left( \prod_{j = 0}^{k - 1} - n - j \right) (- n - k) \frac{d^{m - k - 1} g_{n + k + 1}}{dx^{m - k - 1}}\\ &= \left( \prod_{j = 0}^k - n - j \right) \frac{d^{m - k - 1} g_{n + k + 1}}{dx^{m - k - 1}} \end{align}

Proof for \(k = 1\).

\begin{align} \frac{d^m g_n}{dx^m} &= \left( \prod_{j = 0}^0 - n - j \right) \frac{d^{m - 1} g_{n + 1}}{dx^{m - 1}}\\ &= - n \cdot \frac{d^{m - 1} g_{n + 1}}{dx^{m - 1}} \end{align}

It is proven.

Now there is a definition for the \(m\) derivative for \(g_n\)

\begin{align} \frac{d^m g_n}{dx^m} &= \left( \prod_{j = 0}^{m - 1} - n - j \right) g_{m + n}\\ &= (-1)^{m} \frac{(n + m - 1)!}{(n - 1)!} \cdot g_{m + n} \end{align}

For the \(n\) derivative of \(y\)

\begin{align} \frac{d^2 f}{dx^2} &= g_2(x) & \frac{d^ny}{dx^n} &= \frac{c_y}{c_x} \cdot \frac{d^nf}{dx^n} \end{align} \begin{align} \frac{d^n y}{dx^n} &= \frac{c_y}{c_x} \cdot \frac{d^{n - 2} g_2}{dx^{n - 2}}\\ &= \frac{c_y}{c_x} (-1)^n \cdot (n - 1)! \cdot g_n(x)\\ \end{align}

This formula is only valid for \(n \ge 2\).

And evaluating at \(x = 0\):

\begin{align} f(0) &= c_x & g_n(0) &= - \frac{1}{c_x^{n - 1}} \end{align} \begin{align} \left. \frac{d^n y}{dx^n} \right|_{x = 0} &= \frac{c_y}{c_x} (-1)^n \cdot (n - 3)! \cdot \left( - \frac{1}{c_x^{n - 1}} \right)\\ &= \frac{c_y}{c_x^n} (-1)^{n - 1} \cdot (n - 1)!\\ \end{align}
