Shortcuts

Windows + Space Cycles to next language
Shift + Windows + Space Cycles to previuos language
Alt + Tilde Switchs between kana input and romaji input
Alt + Caps Lock Switchs to katakana (only in kana input)
Ctrl + Caps Lock Switchs to hiragana (only in kana input)
Ctrl + Enter Enters text as it is written (useful to write only hiragana / katakana, avoiding kanji suggestions)
